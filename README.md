# flask-practice
Personal practice repo

## Prerequisite

* python version 3

## Virtual environment

`create a virtual environment called 'my-venv'`

```bash
$ python -m venv my-venv
```

`Dive into 'my-venv'`

```bash
$ source my-venv/bin/activate
```

`Install dependencies`

```bash
$ pip install -r requirements.txt
```

`Check dependencies`

```bash
$ pip list
```

## Execute server

```bash
$ python src/app.py
```
